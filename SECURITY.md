# Security Policy

## Supported versions

Security updates are prioritized for:

- `0.7.x` (active)

Older versions may receive fixes at maintainer discretion.

## Reporting a vulnerability

Please report vulnerabilities privately via GitHub Security Advisories:

- Repository -> Security -> Advisories -> "Report a vulnerability"

If that path is unavailable, open a private communication channel with the
maintainers and include:

- affected version/commit,
- impact and attack scenario,
- reproduction details,
- and any proposed mitigation.

Please do not disclose the issue publicly until a fix or mitigation is ready.
