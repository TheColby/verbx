# Notebook Examples

These notebooks provide minimal, reproducible examples for the stable
`verbx.api` Python surface.

## Included notebooks

- `01_render_and_analyze.ipynb` — render with `render_file` then inspect metrics with `analyze_file`.
- `02_generate_ir.ipynb` — generate/write IR assets with `generate_ir`.
- `03_corpus_pipeline.ipynb` — combine API + CLI calls for dataset workflow orchestration.

## Execution notes

- Requires Python `>=3.11` and package dependencies installed.
- Run from repo root so relative paths resolve to local sample/temp files.
- These are intentionally lightweight examples; for larger experiments, migrate
  cells into your project notebooks and pin versions.
