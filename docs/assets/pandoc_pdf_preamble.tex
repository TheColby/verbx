  \chapter*{About This Edition}
  \addcontentsline{toc}{chapter}{About This Edition}
  This book combines the verbx command-line manual, plug-in architecture handbook,
  production and validation cards, illustrated DSP atlas, impulse-response guidance,
  academic references, and a listening appendix devoted to music in which reverberation
  is part of the composition or production.

  The software is evolving. Statements marked as protocols, foundations, beta behavior,
  or future work are not compatibility certifications. Commands, schemas, figures, and
  generated cards are rebuilt from the same repository so that the book can be audited
  alongside the code.

  \chapter*{Colophon}
  \addcontentsline{toc}{chapter}{Colophon}
  This digital book was generated on \@date\ from Markdown and repository assets by
  \texttt{scripts\_generate\_docs\_pdf.py}. Pandoc 3.8.2 assembled the manuscript;
  XeLaTeX and TeX Live 2025 produced the PDF; MakeIndex generated the back-of-book index;
  and TikZ drew the operational-card illustrations as resolution-independent vectors.

  The text face is TeX Gyre Schola, an open typeface in the New Century Schoolbook
  tradition. Menlo is used for commands and code. The page is U.S. Letter with a book
  hierarchy, roman-numbered front matter, arabic-numbered main matter, lettered appendices,
  and two-column index. Tables use alternating neutral fills; diagrams use the verbx teal,
  gold, rust, blue, plum, and sage palette.

  The illustrated atlas is generated by \texttt{scripts/generate\_userguide\_figures.py}.
  The 588 operational cards are generated by \texttt{scripts\_generate\_plugin\_guide.py}
  and composed one card per page with one of thirteen semantic vector-diagram families.
  The consolidated Markdown remains available at \texttt{docs/USERGUIDE.md}.
