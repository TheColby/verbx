# verbx

**Extreme ambient audio reverberator**

A Python CLI tool for applying extreme reverberation effects to audio files.

## Features

- **Extreme Reverb:** High-quality, long-tail reverberation optimized for ambient music.
- **CLI Interface:** simple command-line interface.

## Installation

```bash
pip install verbx
```

## Usage

```bash
python -m verbx input.wav output.wav
```

## Development

To set up the development environment:

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -e .
   ```
3. Run the CLI:
   ```bash
   python -m verbx --help
   ```
